# Unity 2D 拼面具游戏

## 游戏说明

这是一个2D拼面具游戏，玩家需要将不同形状和颜色的素材拼接到面具的9个位置上。

## 游戏规则

### 面具槽位置（9个）
- 左眼、右眼、左耳、右耳、鼻子、嘴巴、左面纹、右面纹、额饰

### 素材系统
- **形状**：三角形、椭圆形、长方形（3种）
- **颜色**：红、橙、黄、绿、青、蓝、紫（7种）
- **数量**：每种组合有2张，共 3 × 7 × 2 = 42个素材

### 游戏机制
1. **随机抽取**：从素材库中随机抽取素材
2. **放置**：拖拽素材到面具槽的相应位置
3. **暂存**：将素材放入缓存槽（长度可设置，默认5，范围3-10）
4. **丢弃**：丢弃当前素材，抽取下一个
5. **提交**：当所有9个位置都填充完成后，可以提交面具

## 脚本说明

所有脚本位于 `Assets/Scripts/` 目录下：

- `MaterialData.cs` - 素材数据类和枚举定义
- `GameManager.cs` - 游戏核心管理器
- `MaterialLibrary.cs` - 素材库和视觉管理
- `MaterialVisual.cs` - 素材视觉组件
- `MaskSlotManager.cs` - 面具槽管理器
- `MaskSlot.cs` - 单个面具槽组件
- `CacheSlotManager.cs` - 缓存槽管理器
- `CacheSlot.cs` - 单个缓存槽组件
- `DraggableMaterial.cs` - 拖拽功能组件
- `UIController.cs` - UI控制器

## 使用步骤

1. 在Unity中创建新场景
2. 创建Canvas和UI结构
3. 将脚本挂载到对应GameObject上
4. 在Inspector中设置所有引用
5. 运行游戏

详细配置说明请参考各脚本的注释。
